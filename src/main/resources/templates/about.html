What is Shortify?

Shortify is a URL shortening service designed to make long and complex web links manageable and easy to share. Simply paste a long URL into the input box, and Shortify will generate a shorter version that can be easily copied and shared. Additionally, users can track the total number of clicks the shortened URL has received, along with other details such as the time of the last click.

Technologies Used

  • Java Spring Boot - For backend development and URL processing.
  • MongoDB - For storing URL data.
  • Thymeleaf - For rendering dynamic HTML pages.
  • Bootstrap 4 - For responsive design and user interface components.
  • JavaScript - For handling dynamic form submission and frontend logic.
